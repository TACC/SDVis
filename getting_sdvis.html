Getting SDVis

Precompiled Binary Download Packages

Download links for binary releases will be added soon. Until then, please build the SDVis components from their respective sources.

Building SDVis Components from Source

The latest SDVis sources are always available at the repositories listed below. The default "master" branches should always point to the latest tested bugfix release.
  • GLuRay (tacc.github.io/GLuRay/)
  • GraviT (tacc.github.io/GraviT/)
  • Manta (mantawiki.sci.utah.edu)
  • OpenSWR (openswr.org)
  • OSPRay (ospray.github.io)
  • vtkOSPRay (tacc.github.io/vtkOSPRay/)
    • pvOSPRay (tacc.github.io/pvOSPRay/)
    • visitOSPRay (tacc.github.io/visitOSPRay/)
